Guia Básico

Sobre o jogo

Wheelbarrow é um jogo para testar sua habilidade para coletar tijolos! Isso mesmo, coletar tijolos!

Objetivo

O jogador é responsável por controlar uma carriola, e deve coletar a quantidade específica de tijolos de cada fase.

Só não pode deixar a barra de vida zerar!

Controles

  • Mover para a esquerda: Tecla A
  • Mover para a direita: Tecla D
  • Usar "boost": Shift + mover para esquerda ou direita

Como jogar

    Controles

  • Mover para a esquerda: Tecla A
  • Mover para a direita: Tecla D
  • Usar "boost": Shift + mover para esquerda ou direita

Guia avançado

Mecânicas do jogo

O objetivo principal é coletar os tijolos! Desviando dos pesos, e coletando outros itens se necessário!


Existe a barra de peso

Se ela estiver cheia, como na imagem, você não conseguirá usar o seu poder


O peso aumenta conforme você coleta tijolos e/ou pesos



Para reduzir o peso, você deve coletar as penas que aparecem



Barra de vida



Se você deixar a barra de vida esvaziar, você perde!
Para enchê-la você deve coletar a "cura"



Barra de poder



A barra de poder é responsável pelo seu boost de velocidade utilizando o shift!
Para enchê-la, colete os "boosts"



Objetivos

Cada fase tem uma quantia de tijolos como objetivo

Quantidade de tijolos necessária para passar de fase.

Game over

Se sua vida chegar a zero....



No momento o jogo só possui 2 fases, espere a próxima atualização!
